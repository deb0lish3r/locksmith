== Description ==

<p>NIC Photo Editor develop by <a href="http://www.indianic.com/" target="_blank">IndiaNIC</a> allows us to create some absolutely amazing graphics on the web. Merge multiple image and convert in single image. Save your canvas image in media library and you can insert it in any location of your post or page.</p>

<p>Objects can be easily flipped or rotate in any direction. Or locked in movement, scaling, etc.</p>

`Features`
<ul>
<li>Ability to create transparent PNG image.</li>
<li>Option to change canvas background color.</li>
<li>Feature to re-size your canvas.</li>
<li>Ability to set object size by scaling it OR by give width/height of object.</li>
<li>Feature to Bring Front or Send Back to any selected object of canvas area.</li>
<li>Ability to delete any selected object.</li>
<li>You can preview your image by View Image option.</li>
<li>Save/Add your canvas image as media library post.</li>
<li>No Need Of Knowledge of PHP, HTML or CSS. But for those having knowledge can easly modify or customize.</li>
</ul>

== Installation ==

1. Upload the NIC-photo-editor folder to the /wp-content/plugins/ directory `OR` Upload from Plugins > Add New > Upload zip file.

2. Go to the 'Plugins' page of your WordPress administration area and activate the plugin.
